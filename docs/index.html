jQuery Camera Capture

jQuery Camera Capture is a plugin that allows you to easily use the MediaRecorder API to capture video from a user's webcam.

Example

<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <title>Camera Capture Demo</title>
  <style>
    video {
      background:#000;
      display:block;
    }
  </style>
</head>
<body>
  <div id="video"></div>
  <script src="//code.jquery.com/jquery.min.js"></script>
  <script src="jquery.camera_capture.js"></script>
  <script>
    $('#video').cameraCapture({
      uploadURL: "upload.php"
    });
  </script>
</body>
</html>

Settings

Name Type Default Description
width Integer 320 The width of the video being recorded.
height Integer 240 The height of the video being recorded.
uploadURL String empty The URL to POST the video to upon saving.
recordButtonHTML HTML String <button class="camera-capture-record-button">Record</button> The HTML markup for the record button
stopButtonHTML HTML String <button class="camera-capture-stop-button">Stop</button> The HTML markup for the stop button
saveButtonHTML HTML String <button class="camera-capture-save-button">Save</button> The HTML markup for the save button
trashButtonHTML HTML String <button class="camera-capture-trash-button">Trash</button> The HTML markup for the trash button
scaleVideo Boolean True Whether or not to scale the video displayed in the browser up or down.
videoScaleFactor Float 2.0 The factor by which to scale the video if scaleVideo is true.

Public Methods

In addition to creating controls for video recording and saving, their public methods are also exposed.

Name Description
startRecording Start a video stream from the user's webcam and begin capturing it.
stopRecording Stop capturing video from the user's webcam and load it into the preview for playback.
saveRecording Save the video to your server at the url specified by uploadURL
trashRecording Clear the buffer, trash the current video, and reset everything so the user can start over.
getBrowser A simple browser detection method used by the plugin, exposed for your convenience.

Events

The plugin also fires events for almost everything it does. This allows you to do things like gracefully handle errors, togle visability/state of record and save controls, etc.

Name Description
camera_capture.browser_not_supported Triggered if the user's browser does not support the MediaRecorder API
camera_capture.start_recording Triggered when the plguin begins capturing video.
camera_capture.stop_recording Triggered when the plugin stops capturing video.
camera_capture.init_save Triggered when saveRecording is called.
camera_capture.save_success Triggered when saveRecording finishes successfully.
camera_capture.save_failed Triggered when saveRecording fails its AJAX request.
camera_capture.stream_error Triggered when there is a stream error while capturing video.
